## What GeoPilot Includes

- `Map Snapshot` for fast map capture and sharing
- `Export KMZ` for exporting selected map content
- `Geo Report` for geographic reporting workflows
- `Google Maps Link` for generating direct Google Maps URLs from features
- `KMZ to Feature Class` conversion workflow
- `Ain Alabid Offset` and reverse utility tools
- Extra basemap options integrated into ArcGIS Pro

## Why This Repository Looks Simple

This repository is intentionally kept lightweight for end users.

- It provides the installer package directly
- It is designed for quick download and installation
- Source code is not published here

## Installation

1. Close `ArcGIS Pro`
2. Download the installer file:
   [GeoPilot-v1.0.45.esriAddinX](https://github.com/ehis404/Geopilot/releases/download/v1.0.45/GeoPilot-v1.0.45.esriAddinX)
3. Double-click the `.esriAddinX` file
4. Confirm installation
5. Open `ArcGIS Pro`
