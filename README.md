## What GeoPilot Delivers

| Workflow Area | Included Tools | Purpose |
|---|---|---|
| Sharing | `Map Snapshot`, `Export KMZ`, `Geo Report` | Fast sharing and delivery of spatial outputs |
| Conversion | `KMZ to Feature Class` | Convert KMZ/KML content into ArcGIS-ready layers |
| Smart Links | `Google Maps Link` | Generate direct Google Maps links from features |
| Utilities | `Ain Alabid Offset`, `Ain Alabid Reverse` | Apply and reverse custom XY offset workflows |
| Basemaps | Custom Basemap Gallery entries | Extend ArcGIS Pro with additional basemap options |

## Why This Repository Is Clean

This repository is intentionally focused on **installation only**.

- Ready for direct download
- Simple for end users
- No source code published here
- Suitable for sharing the tool without exposing the internal project files

## Installation

1. Close `ArcGIS Pro`
2. Download the installer file:
   [GeoPilot-v1.0.45.esriAddinX](https://github.com/ehis404/Geopilot/releases/download/v1.0.45/GeoPilot-v1.0.45.esriAddinX)
3. Double-click the `.esriAddinX` file
4. Confirm installation
5. Reopen `ArcGIS Pro`
