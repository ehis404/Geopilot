## What GeoPilot Includes

<table>
  <tr>
    <td width="33%" valign="top">
      <h3>Sharing</h3>
      <p><strong>Map Snapshot</strong>, <strong>Export KMZ</strong>, and <strong>Geo Report</strong> help you move map outputs quickly with fewer steps.</p>
    </td>
    <td width="33%" valign="top">
      <h3>Conversion</h3>
      <p><strong>KMZ to Feature Class</strong> and supporting workflows make it easier to bring shared spatial content back into ArcGIS Pro.</p>
    </td>
    <td width="33%" valign="top">
      <h3>Smart Utilities</h3>
      <p><strong>Google Maps Link</strong>, custom basemaps, and utility tools are built for fast GIS work instead of menu-heavy routines.</p>
    </td>
  </tr>
</table>

## Core Toolset

| Area | Tools |
|---|---|
| Sharing | `Map Snapshot`, `Export KMZ`, `Geo Report` |
| Conversion | `KMZ to Feature Class` |
| Spatial Utility | `Google Maps Link`, `Ain Alabid Offset`, `Ain Alabid Reverse` |
| Basemaps | Additional gallery basemap entries |

## Installation

1. Close `ArcGIS Pro`
2. Download [GeoPilot-v1.0.45.esriAddinX](https://github.com/ehis404/Geopilot/releases/download/v1.0.45/GeoPilot-v1.0.45.esriAddinX)
3. Double-click the installer file
4. Confirm add-in installation
5. Open `ArcGIS Pro` again
